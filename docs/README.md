## Adding Todos

Adding a todo task allows you to track tasks that does not have specific deadlines or event time

рҹ’Ў Action: Input the keyword `todo` followed by the task description

рҹҺҜ Outcome: Bambam will add the todo task to the task listрҹ“қ

**Example:** `todo buy bread`

```
Got it! Your task has been added рҹҢё
  [T][ ] buy bread
Now you have 6 tasks in the list. рҹ’–
```

## Adding deadlines

Adding a deadline task allows you to track tasks that should be done before a certain date and time 

рҹ’Ў Action: Input the keyword `deadline` followed by the task description, then followed by the `/by` date and time

рҹҺҜ Outcome: Bambam will add the deadline task to the task listрҹ“қ

**Example:** `deadline finish assignment /by 2025-09-19 2359`

```
Got it! Your task has been added рҹҢё
  [D][ ] finish assignment (by: Sep 19 2025 2359)
Now you have 6 tasks in the list. рҹ’–
```

## Adding events

Adding a events task allows you to track tasks that should be completed within a specific time period

рҹ’Ў Action: Input the keyword `event` followed by the task description, then followed by the `/from` date and time and `/to` date and time

рҹҺҜ Outcome: Bambam will add the event task to the task listрҹ“қ

**Example:** `event project meeting /from 2025-09-19 1800 /to 2025-09-19 2359`

```
Got it! Your task has been added рҹҢё
  [E][ ] project meeting (from: Sep 19 2025 1800 to: Sep 19 2025 2359)
Now you have 6 tasks in the list. рҹ’–
```
