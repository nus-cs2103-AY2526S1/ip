# Sam the Zen Master User Guide

🧘‍♂️ Welcome to Sam the Zen Master - your mindful task management companion. Sam helps you organize your tasks with wisdom, peace, and gentle guidance.

## Getting Started

Launch the application and Sam will greet you with a peaceful welcome message. Type your commands in the text field and press Enter or click Send.

## Commands

### Adding Tasks

**Add a simple task:**

```
todo read a book
```

Sam responds: "🌱 A new intention has taken root in your journey: [T] [ ] [MEDIUM] read a book"

**Add a deadline:**

```
deadline submit assignment /by 2024-12-01
```

Sam responds: "🌱 A new intention has taken root in your journey: [D] [ ] [MEDIUM] submit assignment (by: 2024-12-01)"

**Add an event:**

```
event team meeting /from 2pm /to 4pm
```

Sam responds: "🌱 A new intention has taken root in your journey: [E] [ ] [MEDIUM] team meeting (from: 2pm to: 4pm)"

### Managing Tasks

**View all tasks:**

```
list
```

**Mark task as done:**

```
mark 1
```

**Mark task as not done:**

```
unmark 1
```

**Delete a task:**

```
delete 1
```

**Set task priority:**

```
priority 1 high
```

(Use: low/1, medium/2, or high/3)

**Find tasks:**

```
find book
```

### Exit

```
bye
```

Sam will bid you a peaceful farewell.

## Task Format

Tasks display as: `[Type] [Status] [Priority] Description`

- **Type**: T (Todo), D (Deadline), E (Event)
- **Status**: X (done), space (not done)
- **Priority**: LOW, MEDIUM, HIGH

May your productivity journey be mindful and balanced! 🌸
