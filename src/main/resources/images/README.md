# Images Directory

This directory should contain the following image files for the PoopieMeow GUI:

## Required Images

1. **DaUser.png** - User avatar image (should be approximately 100x100 pixels)
2. **DaDuke.png** - PoopieMeow avatar image (should be approximately 100x100 pixels)

## Image Requirements

- Format: PNG (recommended) or JPG
- Size: Approximately 100x100 pixels
- Background: Transparent or solid color
- Content: Simple avatar/icon representing the user and the chatbot

## How to Add Images

1. Download or create two avatar images
2. Name them exactly as specified above
3. Place them in this directory
4. The GUI will automatically load and display these images

## Alternative

If you don't have images, you can temporarily comment out the image loading lines in the MainWindow.java file to run the GUI without avatars.
