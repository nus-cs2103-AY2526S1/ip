# AI Usage Record

## Tools Used
- **ChatGPT (OpenAI)**
- **Perplexity AI**

## Purpose
I primarily used AI tools to assist with the **GUI development** of the Dobby iP. Specifically, I leveraged them to:
- Explore different options for customizing JavaFX components (e.g., circular icons, alignment of dialog boxes).
- Debug issues related to resource loading and JavaFX configuration.

## Observations
- Using AI tools made it much faster to discover the range of **customizability** in JavaFX, compared to searching through official documentation or trial-and-error.
- ChatGPT was particularly useful for generating **ready-to-use code snippets** that I could adapt directly.
- Perplexity AI was helpful for **summarizing references and examples** from multiple sources, giving me context around alternative solutions.
- Overall, the use of AI reduced the time I spent figuring out GUI behavior and allowed me to focus more on integration and testing.

## Productivity Impact
Using AI reduced the time I spent on GUI development by an estimated **40%**, as I could find solutions and workable snippets much faster than manually researching or experimenting.
