# AI Tool Usage and Observations

Using AI tools, primarily GitHub Copilot, significantly accelerated my development process. 

I found that writing clear and concise header comments was extremely helpful, as it allowed the AI to generate clear and well-described method headers. 

For the reminder function, the AI was excellent at suggesting various implementation approaches, which I then researched further to implement correctly. 

AI also allowed me to learn through examples, like the handling date time format function. When I was unsure of the exact syntax, the AI provided immediate contextual examples. It helped me learn and apply new features correctly without breaking my flow to search through external resources.

Overall, it was a major time-saver for routine coding tasks.
