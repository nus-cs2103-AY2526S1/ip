# Rafayel
> "Your mind is for having ideas, not holding them." - David Allen

Rafayel helps free your mind from remembering every little task.  
It’s designed to be **simple**, **efficient** and **fast**, so you can focus on what truly matters.  

## Why Rafayel?
- 📝 Text-based – lightweight and distraction-free  
- ⚡ Easy to learn – start managing tasks in minutes  
- 🚀 Super fast – optimised for productivity  

## How to Use
1. Download it from [here](#).  
2. `Command + R` to run.  
3. Add your tasks.  
4. Let Rafayel handle the rest 😉  

And best of all — **it’s FREE!**

- [x] ✅ Task management (add, mark, unmark, delete, list)
