## 📋 Yapbot Task Manager

Yapbot is a Java-based task management application optimized for command-line users who value speed and efficiency. It supports structured commands and an optional GUI for improved usability.

### ✨ Features
- ⚡ Fast command-line interface for task management
- 🧠 Command parsing for adding, updating, and deleting tasks
- 🕘 Command history tracking for improved workflow
- 🖥 Optional GUI for visual interaction
- 🧩 Modular and maintainable code structure

### 🛠 Tech Stack
- Java
- Gradle (build automation)
- File I/O for data persistence

### 📜 Core Functionality
- Add, delete, and update tasks
- Set reminders and deadlines
- View task lists with structured formatting
- Maintain command history
