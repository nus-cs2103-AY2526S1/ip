# TaskLynx project template

This is a greenfield Java project. It is a task manager app used for tracking and sorting tasks.

## Downloading and using TaskLynx

Prerequisites: TaskLynx has cross-platform compatibility for Java 17

1. Download the v1.0 release jar file
2. Launch the jar file
3. You're ready to use TaskLynx! Do refer to the online user guide at https://zmzzz3.github.io/ip/.

Be aware that TaskLynx will create a folder "data" containing a text file "log.txt" that is used to store the task list.
